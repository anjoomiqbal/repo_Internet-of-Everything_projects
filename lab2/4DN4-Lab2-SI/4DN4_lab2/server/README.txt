The program has been developed in Java.

To run the code:
	1. create an empty netbeans project 
	2. in the project create a package named 'ftpserver'
	3. in this package copy all the java files in this folder
		FTPserver.java		
		FileSystem.java
		HandleQuit.java
		FTPserverProtocol.java
	4. there are option of putting the port number in argument
	5. if no port number is provided in the argument 
           then the program will ask for port number
