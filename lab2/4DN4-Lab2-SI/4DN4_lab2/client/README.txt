The program has been developed in Java.

To run the code:
	1. create an empty netbeans project 
	2. in the project create a package named 'ftpclient'
	3. in this package copy the only java file in this folder
		FTPclient.java	
	4. there are option of putting the Server IP address and port number in argument
	5. if they are not provided in the argument 
           then the program will ask for them
